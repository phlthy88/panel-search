# Initial Concept
A modern search widget for GNOME Shell 48-50 that integrates into the top panel, similar to Windows 11 taskbar search.

# Product Guide: Panel Search

## Overview
Panel Search is a GNOME Shell extension designed to unify system search, web search, and basic utility functions into a single, intuitive interface. It aims to provide a centralized search experience that is both powerful for efficiency-minded users and accessible for general GNOME users.

## Target Users
- **General GNOME Users:** People seeking a familiar, taskbar-style search interface.
- **Efficiency-focused Users:** Those who prioritize fast, keyboard-centric workflows for launching apps and searching the web.

## Key Goals
- **Comprehensive Search:** Merge application, settings, and web search into one intelligent omnibox.
- **Seamless GNOME Integration:** Ensure the widget looks and feels like a native part of GNOME Shell.
- **User-Centric Ranking:** Prioritize search results based on frequent usage to learn from user behavior.

## Core Features
- **Unified Omnibox Search:** A single entry point for all search sources (apps, settings, files, web).
- **Built-in Utilities:** Native support for basic math, unit conversions (distance, weight, volume), and web search (Google, DuckDuckGo, Bing).
- **Predictive Search:** Intelligent ranking that highlights frequently used items.
- **Fuzzy Search:** Resilient search logic for apps and settings.

## Constraints & Requirements
- **GNOME HIG Compliance:** Adhere strictly to GNOME's Human Interface Guidelines for visual consistency.
- **Version Compatibility:** Maintain full support for GNOME Shell versions 48, 49, and 50.
- **Resource Efficiency:** Maintain a lightweight memory and CPU footprint to ensure a smooth desktop experience.
